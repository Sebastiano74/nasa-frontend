# NASA NEO Dashboard - Frontend

Dashboard per esplorare gli asteroidi in avvicinamento alla Terra. Dati reali dalla NASA.

## Tecnologie
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Shadcn UI
- Recharts
